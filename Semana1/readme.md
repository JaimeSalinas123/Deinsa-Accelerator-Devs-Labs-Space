## 📘 **Contenido de la Semana 1**

### 🔹 Variables y Tipos de Datos

```jsx
let nombre = "Juan";
const edad = 25;
let activo = true;

```

### 🔹 Condicionales

```jsx
if (edad >= 18) {
  console.log("Es mayor de edad");
}

```

### 🔹 Ciclos

```jsx
for (let i = 0; i < 5; i++) {
  console.log("Número:", i);
}

```

### 🔹 Funciones

```jsx
function sumar(a, b) {
  return a + b;
}

const restar = (a, b) => a - b;

```
